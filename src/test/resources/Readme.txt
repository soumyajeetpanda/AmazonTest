####INTELLIJ INSTALLATION######
1. Download the Intellij- Community Edition Installer
2. Launch Intellij
Once the installation is complete you can now launch Intellij. The Intellij Installer has done it's work. Happy coding.

#####JDK INSTALLATION#####
1. Install the JDK package
2. Check the java version loaded in cmd by going to CMD and type java -version
3. If the package is latest, we are good.

####RUNNING THE CODE#####
Steps to run the automation.
1. Import the project as existing project into workspace.
2. Right click on project and run as maven test./Right click on the TestNG.xml file under src-Test-java-resources-TestNG.xml
3. Check the extent report folder for final report generation.
